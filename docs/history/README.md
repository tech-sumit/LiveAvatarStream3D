# Historical design docs

GPU talking-head rendering, realtime streaming, and related paths), kept for provenance.
They predate the 2026-06-26 consolidation and no longer describe this repo. The current
3D-browser-only studio is described by the root [README](../../README.md) and the specs in
[`docs/specs/`](../specs/) (when specs disagree, the newest wins).
